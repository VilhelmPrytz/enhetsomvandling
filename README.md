Source code of [enhetsomvandling.se](https://enhetsomvandling.se). Built using React.

## Running locally

Install npm dependencies.

```bash
npm install
```

Run the development server.

```bash
npm start
```

## Building

Build the app for production.

```bash
npm run build
```
